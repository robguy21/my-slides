A brief history

  1. JavaScript was first created 1995 in { 10 } days by { Brendan Eich }.
  2. Which popular browser at the time was the main driving force behind javascript? { Netscape }
  3. What year did Javascript officially become standardized? { 1997 } It's called ECMAScript because Javascript was trademarked and apparently the alternatives were shit.
  4. Which organization was responsible for standardizing Javascript? { ECMA }
  5. What year did ES2 released? { 1998 } fun fact - ES2 still didn't ship with exceptions, this meant the interpreter was left to handle any and all errors.
