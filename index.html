A brief history

  1. JavaScript was first created 1995 in { 10 } days by { Brendan Eich }.
  2. Which popular browser at the time was the main driving force behind javascript?
  3. Netscape
  4. Javascript finally saw some standardization in { 1997 } from ECMA with the release of ECMAScript 1.0 It's called ECMAScript because Javascript was trademarked and apparently the alternatives were shit.
  5. ECMAScript 2 was finally released in { 1998 } to standardize ECMA with ISO. ES2 still didn't ship with exceptions, this meant the interpreter was left to handle any and all errors.

The ECMA committee gathered once more. Representatives from 4 major companies ({ Adobe, Mozilla, Opera, and Microsoft }) set about agreeing on the feature set for ES4.

ECMAScript - 4 - The fuckit days ( continued ).

With the feature set ready enter our hero - Douglas Crockford, the representative from ({ Yahoo }).
He came out gunning calling the features as essentially bullshit and trying to expand the language too quickly too early. Behind the scenes the ({ Microsoft }) agreed with him but couldn't say anything outright as his company would get flamed for being anti-competitive behaviour ( which would've been totally valid at the time ).

After being convinced by Crockford, Microsoft began aggressively disagreeing with the design choices. Stating that they would refuse to implement the new feature set in their browsers.
Microsoft threatened legal action if the draft was finalized in it's current state. This led to two different drafts being prepared - ES3.1 ( backed by Crockford and Microsoft ) and ES4 ( pretty much backed by everyone who was left disagreeing with Crockford and Microsoft )

ECMAScript - 4

After ({ 8 }) years, the ES4 proposal was officially scrapped as it took far too long the complicated feature set.
